Endpoints/Reference

All requests are sent by a POST request. GET requests are not used and will result in a 403 Forbidden error. Simply type after the url /user/yourusernamehere and parse the JSON. All requests must also have a valid API Key attached in the headers.

/user/< username >
Returns all data about a user. Includes posts, repls, upvotes, upvotes on the repls, etc.
/board/share
Returns data from the specified board. "share" can be replaced with "ask" or "all" for different boards.
